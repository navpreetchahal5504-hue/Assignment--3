Assignment Three - Debug This JS

This repository contains the solution for “Assignment Three - Debug This JS!”
The assignment focuses on debugging JavaScript to fix four broken features on a webpage.

Features

1. Greeting Button
  * Displays a greeting message when clicked.
2. Theme Toggle
  * Switches the page between light and dark mode.
3. Live Input
  * Updates text dynamically as you type.
4. Welcome Form
  * Accepts user input and displays a welcome message.

How to Use

1. Open index.html in your browser.
2. Test each feature:
  * Click the “Say Hello” button
  * Toggle the theme using “Dark Mode” button
  * Type in the input box to see live output
  * Submit your name in the welcome form

Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

Fixes and Notes

* Corrected selector errors in JavaScript
* Fixed event listeners for live input and form submission
* Updated CSS class names to match JS
* Added comments explaining all fixes

Optional Enhancements

* Dynamic button text for theme toggle
* Input validation for empty names
* Live input styling based on length
